Projects

  • Free AI Meal Planner Python | Streamlit | Ollama (Llama 3.2)

    This project is a recipe-generation web app that turns leftover fridge ingredients into complete meal ideas using a locally running large language model. Built with Python and Streamlit, the interface lets users enter a comma-separated list of ingredients, then sends a structured prompt to an Ollama-hosted Llama 3.2 model running on the user's machine. The model returns a fully formatted recipe, including a title, prep and cook time, ingredient list, and step-by-step instructions. The app adds error handling for connectivity and timeout issues, ensures that every user-provided ingredient is explicitly used in the generated recipe, and presents the output in a clean, scrollable UI. Because all inference happens locally through Ollama, the planner requires no API keys, incurs no usage costs, and keeps all user data on-device.

  • Facial Recognition Attendance System Python | OpenCV | Face_recognition

    This project centered on building a camera-based attendance platform that automatically detects and recognizes faces in real time and logs entries to structured CSV records for easy analysis. Using Python with OpenCV and a facial-embedding pipeline, the system captures frames from a live video stream, detects faces, and matches them against a pre-enrolled dataset to identify students or employees. Attendance events are appended to a date-specific CSV file with names and timestamps. Robust preprocessing, confidence-threshold tuning, and duplicate-detection logic significantly reduced false positives and ensured reliable recognition performance across different lighting conditions and viewing angles.

  • Intelligent Autonomous Vehicle Control System C | C++ | STM32 | Arduino | Raspberry Pi | ROS | Linux | CAN Bus | I2C | SPI | PWM | Oscilloscope Debugging

    This project focused on building a robust safety layer for an autonomous vehicle, with an emphasis on manual override and fault tolerance. I developed an RC override system integrating STM32, Arduino, and Raspberry Pi platforms, using I2C, CAN Bus, SPI, and PWM control to ensure smooth transitions between autonomous and manual modes. The embedded software was written in C/C++ on a Linux environment and integrated with ROS for orchestration, diagnostics, and system monitoring. Extensive hardware debugging with oscilloscopes and signal multiplexing techniques significantly improved communication reliability, reduced latency, and increased overall system stability, resulting in a highly responsive and dependable fail-safe control system.

  • Optimized Full-Stack Web Platform with Caching HTML | CSS | JavaScript | React | PHP | SQL | Apache | XAMPP | Git

A major focus of the project was performance optimization, where I implemented server-side caching strategies and backend optimizations to significantly reduce page load times under high-traffic conditions.   • High-Speed FPGA Cache Memory Architecture Verilog | VHDL | C | Zynq-7000 FPGA | Vivado | Hardware Simulation | Debugging

    This project involved designing a custom cache memory architecture on a Zynq-7000 FPGA to improve data access efficiency in high-speed processing systems. Using Verilog, VHDL, and C, I implemented multiple caching strategies, including Least Recently Used (LRU) and direct-mapped caches, to optimize memory hierarchy performance. Vivado was used extensively for synthesis, simulation, and hardware debugging, allowing detailed performance analysis and bottleneck identification. Through targeted hardware-level optimizations, the system achieved significant gains in data retrieval speed and processing throughput.

  • Embedded Autonomous Snow Plough Control System Embedded C | RTOS | Sensors | Motor Drivers | Serial Communication | LTSpice

    As the project lead, I oversaw the complete hardware-software co-design of an autonomous snow plough system. The system was built using Embedded C and integrated sensors, motor drivers, and an RTOS to support real-time obstacle detection and task scheduling. I implemented serial communication protocols for reliable real-time data transfer and contributed to an intelligent path-planning approach based on dataset analysis for predictive navigation. Electrical components and circuit behavior were extensively simulated using LTSpice, followed by coordinated hardware integration and testing to validate system reliability.

  • IoT Remote Monitoring and Control System Arduino | Raspberry Pi | Python | PHP | SQL | React | Kotlin | OpenCV | APIs | I2C | Serial Communication

    This project centered on building an end-to-end IoT monitoring solution using Arduino and Raspberry Pi for real-time data acquisition via I2C and serial communication. I developed a web-based dashboard using PHP, SQL, React, and API integrations to visualize sensor data and system status. In parallel, I built a Kotlin-based mobile application leveraging OpenCV for real-time image processing, enabling remote monitoring and control. Backend data processing and analytics were implemented in Python, improving alerting mechanisms, response times, and overall system uptime through efficient power management and signal multiplexing.
