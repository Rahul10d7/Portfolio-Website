This repository contains the source code for my personal portfolio website.  
It showcases my background, education, skills, projects, and contact information in a clean, responsive single-page layout.

## Features

- **Hero section** with a full-width background image and introductory tagline  
- **Education** section with university logos and degree details  
- **Skills** section using pill-style tags grouped by category  
- **Projects** section with clickable items that reveal detailed descriptions  
- **Experience** summary and **Download Resume** button  
- **Contact** section with icons for phone, email, and LinkedIn

## Tech Stack

- HTML5  
- CSS3 (custom styling, flexbox, hover effects, simple animations)  
- JavaScript (smooth scrolling, typewriter effect, project detail toggles)
